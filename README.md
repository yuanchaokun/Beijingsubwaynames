# 北京地铁站名探索

一个有趣的北京地铁站名网站，收录了300+个地铁站名的起源和历史故事。

## 功能特色

- 🎲 **随机站点** - 随机探索不同的地铁站
- 🚇 **选择线路** - 按线路浏览所有站点
- 🔍 **搜索站点** - 快速搜索感兴趣的地铁站
- 📖 **站名故事** - 了解每个站名的起源和历史
- 🗺️ **地图展示** - 在地图上高亮显示站点位置

## 技术栈

- HTML5 + CSS3 + JavaScript
- 响应式设计，支持移动端
- SVG地图交互
- 搜索联想功能
